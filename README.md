# Gotstl.com points to Azure static page

## Tech
Angular 15


## Limitation
need to have `www.` in front [http://www.gotstl.com](http://www.gotstl.com)

hosted on azure free static pages
 - Angular framework
 - azure functions

Neeed to run for local development

```
npm -prefix=api start
```

1. Install the npm packages

```bash
npm install
```

2. Run the app

```bash
npm start
```


### Deployment 
* via `Github Actions` to  `Azure Static Web Apps CI/CD`
